# ViewHub

<p>This repo contains the code for the [ViewHub]().</p>



- [Live Demo]()

## Usage

<p>ViewHub is a feedback application. Users can easily communicate and provide feedback as well as upvote popular suggestions.</p>
<p>- Enjoy! 🏝️</p>

### Installation

```
npm install
```

```
composer install
```

```
cp .env.example .env
```

```
php artisan key:generate
```

```
php artisan migrate
```

```
npm run dev
```

```
php artisan serve
```
